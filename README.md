# FileCleanUp


## Introduction
Drive full again? Can't run scripts or create new files?
Never fear File Clean Up is here!

This is a python program that takes a folder root and creates a csv file with the file path, size, and the date it was last modified.

# How To
1. Run this in the terminal on the workbench.
`git clone https://gitlabstats-prd/prices/filecleanup/`

**If VS Code**

2. Open `main`.
3. Change the `root` variable to the folder you want to scan.
4. Change the `output_path` to where you want to save the output.
5. Click the `Play Arrow`

**If RStudio**

2. Make sure `reticulate` is installed.
3. Open `main`.
4. Change the `root` variable to the folder you want to scan.
5. Change the `output_path` to where you want to save the output.
6. Click Source or `Ctrl + Shift + S`
